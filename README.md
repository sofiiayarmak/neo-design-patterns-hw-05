# neo-design-patterns-hw-06

1. Реалізовано консольну утиліту для аналізу файлової системи.
2. Було використано два патерни: фасад та адаптер.
3. `ReportManager` керує всім процесом. Всередині використовує `AnalyzerFacade`, який з'єднує аналізатор з потрібним форматом виводу.
4. Кожен формат (JSON, CSV, XML) реалізований як окремий клас з однаковим інтерфейсом.
